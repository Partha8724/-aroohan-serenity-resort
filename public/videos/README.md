# Videos Directory

Please place your video files in this directory:

1. **`hero.mp4.mp4`** (Home Screen Background)
   - Plays in background loop of the Hero section.
2. **`intro.mp4`** (Framed Intro Video)
   - Plays in the framed center of the loading preloader.
   - Recommended resolution: 720p or 1080p, highly compressed for rapid loading.
